# Full Stack E-commerce Project

## Overview
- Full stack e-commerce application
- Backend: Spring Boot
- Frontend: React (Vite)

## Project Structure
- backend → Spring Boot APIs  
- frontend → React Application  

## Tech Stack
- React (Frontend)
- Spring Boot (Backend)
- REST APIs
- MySQL

## Features
- User Signup (In Progress)
- User Login (In Progress)
- Routing (Home, Login, Signup)
- Basic UI Setup

## How to Run

### Backend
- Run Spring Boot application
- Runs on: http://localhost:8080

### Frontend
- cd frontend
- npm install
- npm run dev
- Runs on: http://localhost:5173
